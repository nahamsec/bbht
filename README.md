# Bugbounty Hunting Toolkit

This Repo is the fork of Nahamsec Bohr where I have updated each and every tools for everyday use and up to date

# Installing
- git clone https://github.com/nahamsec/bbht.git
- cd bbht
- chmod +x install.sh
- ./install.sh
